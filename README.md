# cube_problem_console

### esy


```console
npm i -g esy
```

Or, if you're using yarn

```console
yarn global add esy
```

## Up and running

Install the dependencies with esy:

```console
esy
```

Build the project:

```console
esy build
```

Run the example binary:

```console
esy x cube_problem_console < file.name
```
